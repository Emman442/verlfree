# **App Name**: VeriFree

## Core Features:

- Job Management & Posting: Clients can post detailed job listings with success criteria, budget, and deadlines. Freelancers can browse, apply for, and manage active jobs through dedicated dashboards.
- Deliverable Submission & Escrow: Freelancers can submit work deliverables and clients deposit funds into a secure escrow. Milestone-based payments are supported for larger projects.
- Intelligent Contract AI Verification Tool: An AI-powered tool (via GenLayer integration) automatically verifies submitted deliverables against pre-defined success criteria to ensure quality and trigger payment release.
- Automated Payments & Dispute Management: Instant and automated payment release upon successful AI verification. A dedicated flow for freelancers to challenge AI rejection provides clear status and reasoning.
- Wallet-Based User Authentication & Profiles: Secure authentication linked to wallet addresses, allowing users to choose Client or Freelancer roles and build comprehensive profiles with reputation scores and a verified job portfolio.
- Real-time Communication & Notifications: Integrated real-time chat for seamless client-freelancer interaction and dynamic notifications for all key platform activities and status updates.
- Public Job Board & Leaderboards: A filterable public job board for discovering work and transparent leaderboards showcasing top performing freelancers and clients.

## Style Guidelines:

- Primary color: Electric blue (#3B82F6) for interactive elements and highlights, conveying modernity and trust.
- Background color: Deep navy (#07090F), creating a professional and dark theme base that matches the user's explicit request for a dark background.
- Accent color: Lighter purple (#BB85FF) for subtle gradient highlights, adding a touch of sophistication and depth to the dark theme.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and confident aesthetic, suitable for large headings and clear body text, ensuring readability across all content.
- Minimal, clean, and modern line-based icons for a sophisticated visual language. Logo includes a small shield, symbolizing trust and security. Success states feature clean checkmark animations.
- A clean, minimal, and premium layout inspired by Linear and Stripe. Features sharp edges with occasional soft rounded cards, a fixed top navigation bar, and responsive designs that prioritize mobile-first usability while excelling on desktop. Uses two-column layouts and grids effectively.
- Subtle and purposeful Framer Motion animations. Includes smooth fade-ins for page transitions, 20px slide-up with staggering for card mounts, subtle scale effects (1.02 hover, 0.98 press) for buttons, and skeleton loaders for all loading states. Numbers animate up, and toasts slide in from the top right.
